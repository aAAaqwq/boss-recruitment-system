# 前端按钮说明

## 一、顶部认证栏

| 按钮 | 操作 | 后端接口 |
|------|------|---------|
| **登录** | JWT 登录（用户名+密码） | `POST /api/auth/login` |

## 二、VNC 远程桌面工具栏

| 按钮 | 功能 |
|------|------|
| 🔄 刷新 | 刷新 noVNC 连接 |
| 🔍+ 放大 | VNC 画面放大 |
| 🔍− 缩小 | VNC 画面缩小 |
| 📷 截图 | 保存当前 VNC 画面截图 |

## 三、浏览器控制区

| 按钮 | 功能 | 后端接口 |
|------|------|---------|
| 🚀 **打开BOSS** | 连接 Chrome CDP → 导航到 BOSS直聘 → 注入 Cookie → 等待扫码 | `POST /api/browser/open-boss` |
| 🌐 **连接桌面** | 仅连接 VNC 远程桌面，不操作浏览器 | VNC WebSocket |
| ✅ **检查登录状态** | 检测 BOSS直聘是否已登录（Cookie + 页面文案） | `GET /api/browser/check-login` |
| 💾 **导出Cookie** | 将当前浏览器 Cookie 导出到 `/app/data/cookies.json` | `POST /api/browser/export-cookies` |
| 📂 **导入Cookie** | 从 `/app/data/cookies.json` 恢复登录态 | `POST /api/browser/import-cookies` |

### 操作顺序

```
① 连接桌面 → ② 打开BOSS → ③ VNC中扫码登录 → ④ 检查登录状态 → ⑤ 导出Cookie（备份）
```

## 四、批量自动化区（核心功能）

每个按钮前都有一个数字输入框控制**本次处理人数上限**。

| 按钮 | 工作流 | 流程简述 | 后端接口 |
|------|--------|---------|---------|
| 👋 **筛选+打招呼** | F5 | 推荐牛人页 → iframe提取卡片 → 学历/学校/年限筛选 → CDP点击打招呼 → DB记录 | `POST /api/filter/contact` |
| 📄 **批量获取简历** | F6 | 聊天页 → 点"沟通中"筛选 → 逐次点击联系人 → 找附件简历 → 下载/请求 → CDP拦截保存 | `POST /api/resume/batch` |
| 💬 **批量回复未读** | F7 | 聊天页 → 点"未读"筛选 → 逐次点击联系人 → 提取消息 → DeepSeek生成回复 → 回车发送 | `POST /api/workflow/reply-messages` |

### 输入框默认值

| 输入框 | 默认值 | 后端参数 |
|--------|--------|---------|
| 打招呼人数 | 20 | `batch_limit`（单次上限，受 `daily_cap=80` 约束） |
| 获取简历人数 | 10 | `limit` |
| 回复消息人数 | 15 | `max_count` |

## 五、筛选条件（F5 专用）

| 配置项 | 说明 |
|--------|------|
| 学历要求 | 不限 / 本科 / 硕士 / 博士 / 本科及以上 |
| 学校白名单 | 多行文本，一行一个校名（支持 985/211/双一流缩写） |
| 包含关键词 | 候选人简介中必须包含的关键词（一行一个） |
| 排除关键词 | 候选人简介中包含任一关键词则跳过 |
| 保存筛选配置 | 持久化到服务器 |

## 六、话术模板

| 配置项 | 说明 |
|--------|------|
| 初次招呼 | 首次打招呼的话术（支持 `{name}` 变量） |
| 跟进话术 | 二次跟进回复的话术 |
| 自定义模板 | F7 批量回复的兜底模板（AI 失败时使用） |
| 保存话术配置 | 持久化到服务器 |

## 七、岗位模板

可预设多个岗位的公司背景和职位描述，切换岗位后 AI system_prompt 会自动更新。

| 操作 | 说明 |
|------|------|
| 💾 保存 | 保存当前岗位模板 |
| 🗑️ 删除 | 删除选中的岗位模板 |

## 八、快捷导航

| 按钮 | 功能 |
|------|------|
| 💬 聊天页 | 浏览器导航到 BOSS直聘聊天页 |
| 📋 候选人 | 浏览器导航到 BOSS直聘推荐牛人页 |

## 九、底部日志面板

| 按钮 | 功能 |
|------|------|
| 清除 | 清空当前日志 |
| ▲ 展开 / ▼ 收起 | 切换日志面板高度 |

## 十、任务状态栏

页面底部始终显示当前任务状态（等待启动 / 运行中 / 已完成），任务运行时右侧出现 **⏹ 取消** 按钮，点击可中止当前任务。
