# BOSS直聘完整自动化流程 - 运行说明

## 🚀 快速开始

### 1. 打开BOSS直聘网页
在Chrome浏览器中打开BOSS直聘，登录你的账号。

### 2. 运行完整自动化脚本

**在新终端中运行**（因为需要屏幕录制权限）：

```bash
cd ~/.openclaw/workspace/agents/workspace-xuanyuan/boss-recruitment-system
python3 run_full_automation.py
```

## 📋 完整流程说明

脚本会自动执行以下6个步骤：

### 步骤1: 点击"推荐牛人"
- 在左侧导航栏查找"推荐牛人"按钮
- 自动点击进入推荐页面

### 步骤2: 点击"筛选"
- 在右上角查找"筛选"按钮
- 自动点击打开筛选面板

### 步骤3: 勾选筛选条件
- 自动勾选"985"
- 自动勾选"211"
- 自动勾选"本科"
- 自动勾选"3年以上"

### 步骤4: 点击"确定"
- 在筛选面板中查找"确定"或"确认"按钮
- 自动点击应用筛选条件

### 步骤5: 扫描候选人
- 扫描候选人列表
- 识别所有"打招呼"/"立即沟通"/"继续沟通"按钮
- 提取候选人信息

### 步骤6: 点击"打招呼"
- 自动点击每个候选人的"打招呼"按钮
- 测试阶段限制5个候选人
- 每次点击间隔0.8秒，避免操作过快

## ⚙️ 配置说明

### 修改联系人数限制

编辑 `run_full_automation.py`，找到：

```python
max_contacts = 5  # 测试阶段限制5个
```

修改为你想要的数量（建议不超过80）。

### 修改筛选条件

编辑 `run_full_automation.py`，找到步骤3的部分，可以添加或删除筛选条件：

```python
# 勾选985/211
find_and_click("985", region=filter_region, confidence=40.0)
find_and_click("211", region=filter_region, confidence=40.0)

# 勾选本科
find_and_click("本科", region=filter_region, confidence=40.0)

# 勾选3年以上
find_and_click("3年", region=filter_region, confidence=40.0)
```

### 修改OCR置信度

如果识别不准确，可以降低置信度阈值：

```python
confidence=40.0  # 降低到30.0或20.0
```

## 🐛 故障排除

### 问题1: 找不到"推荐牛人"
- 确保BOSS直聘页面已完全加载
- 确保左侧导航栏可见
- 尝试降低置信度到30.0

### 问题2: 找不到"筛选"
- 确保已进入推荐页面
- 确保右上角筛选按钮可见
- 尝试扩大搜索区域

### 问题3: 找不到筛选选项
- 确保筛选面板已打开
- 确保筛选面板在屏幕中央偏右
- 尝试调整 `filter_region` 的坐标

### 问题4: 屏幕录制权限
如果提示权限错误，请：
1. 打开"系统偏好设置" → "安全性与隐私" → "屏幕录制"
2. 勾选"终端"或你使用的终端应用
3. 重启终端
4. 重新运行脚本

## 🔧 高级功能

### 1. 调试模式

如果需要查看每一步的截图，脚本会自动保存：
- `debug_推荐牛人.png` - 步骤1截图
- `debug_筛选.png` - 步骤2截图
- `debug_985.png` - 步骤3截图
- `candidates_scan.png` - 步骤5截图

### 2. 自定义区域

如果默认区域不准确，可以修改：

```python
# 左侧导航栏区域
left_sidebar = (0, 80, 140, 460)

# 右上角区域
top_right = (1400, 150, 400, 150)

# 筛选面板区域
filter_region = (800, 200, 600, 600)

# 候选人列表区域
candidates_region = (200, 200, 1400, 800)
```

### 3. 添加日志记录

脚本会自动打印详细日志，包括：
- 每一步的执行状态
- OCR识别结果
- 点击坐标
- 错误信息

## 📝 注意事项

1. **测试阶段**：建议先用小数量（5个）测试，确认流程正常后再增加
2. **速度控制**：每次点击间隔0.8秒，避免被BOSS检测为机器人
3. **人工监督**：建议在旁边观察，遇到问题随时按 `Ctrl+C` 中断
4. **每日限制**：BOSS直聘有每日联系上限，建议不超过80人
5. **账号安全**：避免频繁操作，保持自然的使用节奏
